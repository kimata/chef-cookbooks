nas Cookbook
============
This cookbook configures a server as NAS.

Requirements
------------
Assume Ubuntu 12.04 or later.

Attributes
----------
None.

Usage
-----

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[nas]"
  ]
}
```
